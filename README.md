# Grant - Educational Platform

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Run the development server:
```bash
npm run sim
```

3. Open

- Educational platform interface
- Group management
- Task management
- Code editor integration
- Scheme editor
- Redux state management
- SCSS styling
